# Garden Related - Simple Structural Parts

## Rain Barrel Adapter

```
RainBarrelAdapter01.FCStd
```

This is a simple adapter for rain barrels (parameterized, defaulting to a standard
chemistry barrel. The adapter has insets for two M4 nuts on each arm to mount
fixtures or pumps as well as sensors, a small hole for pipes, a medium hole for
pipes or ultrasonic level sensors and a central hole for pipes.
