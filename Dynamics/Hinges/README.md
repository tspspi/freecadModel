# Hinges

## Print-in-place template

![](https://raw.githubusercontent.com/tspspi/freecadModel/master/Dynamics/Hinges/PrintInPlace01__001.png)

A parametric, print-in-place hinge template designed for easy customization. The model is driven via the spreadsheet, allowing quick adjustments to fit different dimensions and tolerances. For best results, print it flat on the build plate.

## Aluminum profile (20B) 2mm plexi hinge

![](https://raw.githubusercontent.com/tspspi/freecadModel/master/Dynamics/Hinges/Profile_Plexi_Hinge_001__001.png)

A simple hinge for mounting a 2mm plexi (acrylic) panel to the outside of a frame built from 20x20 B-type aluminum extrusion. Designed to be fastened using M4 screws.
